# Changelog

## [1.0.0] - 2026-04-13

### Added
- Initial release of YEMA-CYBER Security Bot
- Voice greeting on application startup
- ASCII art logo display
- Personalized user interaction with name input
- Cybersecurity responses for:
  - Password safety tips
  - Phishing awareness
  - Safe browsing practices
  - Two-factor authentication (2FA)
  - General cybersecurity advice
- Colorful console UI with typing effect
- Input validation for empty responses
- Exit commands (goodbye, quit, exit)

### Technical Features
- Built with C# .NET 6.0
- Modular code structure with separate classes
- GitHub Actions CI/CD pipeline
